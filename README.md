# How to use

The project can be ran either locally or in a docker container.

## Docker

1. Clone the repo.
2. Run `docker-compose up`.

## Locally

1. Clone the repo open the `Database.php` file.
2. Change `host` to `localhost` and `password` to `''`.
3. Run `npm start`.
